# Student Record Management System

Description

A simple Java application developed to manage student records using arrays.  
The program allows users to input student names and marks, display records, and perform basic calculations.

Features

1.  Add student names and marks
2.  Display all student records
3.  Calculate average marks
4.  Identify highest scoring student

Technologies Used

1. Java
2. Arrays
3. Loops
4. Conditional statements
5. Scanner class

Objectives

This project was built to strengthen my understanding of:
1. Array handling in Java
2. Loop structures
3. Basic input/output operations
4. Program structure and logic building
